# sike.pona.la: the Toki Pona webring

A webring is a collection of related websites connected by a common navigation bar which allows users to cycle through members’ sites. 

With time, webrings became obsolete, but they are still widely utilized by the “indieweb” movement. This one is for [Toki Pona](https://tokipona.net/) speakers.
